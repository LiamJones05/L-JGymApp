# L-JGymApp
Repository to support the development of Liam and James' gym app project


POSTMAN request formats:

Create new user:
{
	"email" : "Example@example.com",
	"password" : "ExamplePassword123"
}


Create new workout:
{
    "workoutName": "Example Name",
    "workoutDate": "20XX-MM-DD",
    "userId": X
}
